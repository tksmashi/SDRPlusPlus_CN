## 修复中文显示问号问题 (commit 09b6c0b7)
- 根因1: MSVC 未加 /utf-8 编译选项，UTF-8 中文字符串被按 GBK 解读 → 乱码
- 根因2: Roboto-Medium.ttf 无 CJK 字形，glyph ranges 无中文范围 → 问号
- 修复1: CMakeLists.txt MSVC flags 加 /utf-8（Debug + Release 两处）
- 修复2: style.cpp loadFonts() 添加 CJK 支持:
  - baseBuilder.AddRanges(GetGlyphRangesChineseSimplifiedCommon())
  - findCJKFont() 自动检测系统 CJK 字体（Windows/macOS/Linux 多路径回退）
  - ImFontConfig MergeMode=true 将 CJK 字形合并到 Roboto baseFont
- Windows 优先使用 msyh.ttc(微软雅黑)，已验证本地4个中文字体均可用
- 待手动 git push
